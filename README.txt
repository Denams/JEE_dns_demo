Projet réalisé en JEE sous eclipse reprenant le principe d'un "dns" avec une vision bien plus simplifié.
Le client choisis un nom de domaine dans une liste, et le valide. Ce choix est envoyer par une requête HTTP 
qui sera utilisée par le serveur pour interroger une base donnée et ainsi retourner au client l'ip du domaine souhaité.

Projet hébergé sur : http://cascadone.net/dns_demo/dns
